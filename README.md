# Library Management System

## 📌 Overview
This project is a Library Management System designed to automate and simplify library operations using Python and MySQL.

## 🚀 Features
- Add new books to the database  
- Issue books to users  
- Return books and update availability  
- Delete outdated or damaged books  
- Efficient CRUD operations  

## 🛠️ Tech Stack
- Python  
- MySQL  
- PyMySQL  

## 🗄️ Database Design
- **booklist table**: Stores book details (ID, title, author, status)  
- **issue table**: Stores issued book records  

## ⚙️ Functionality
The system performs Create, Read, Update, and Delete (CRUD) operations on a relational database to manage library records efficiently.


## ▶️ How to Run
1. Install MySQL and create database  
2. Update database credentials in code  
3. Run the Python file
